# Distributed-Delay-Stabilizes-Bistable-Gene-Networks
## Version information and Requirements
Python 3.13.9

Numpy 2.3.1

Numba 0.61.2

Matplotlib 3.10.3

Seaborn 0.13.2

Pandas 2.3.1

Writen and run exclusively in Linux and Mac operating systems. File management may fail on Windows. 

## Organization of this Repository
The "Delayed-Gillespie" folder contains code to simulate and sample trajectories from the models in the paper. Data generation is ruinously slow, however, due to waiting for sufficiently many rare events to take place. 

The "Figure-Data" folder contains code and raw/post-processed data to recreate the figures found in the paper.
